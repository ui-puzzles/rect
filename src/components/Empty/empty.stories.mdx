## Empty

Display fridendly tips quickly when a page or container have no data provided. 

## Examples

There are some examples as follows which can help you get started quickly.

### Default

As follows, the basic usage is so easy.

<Canvas>
  <Story name="Default">
    <Divider dashed textStyle={{ color: '#999' }}>default</Divider>
    <Empty />
  </Story>
</Canvas>
