## PageHeader

A page header with some common actions. Usually can be used to display basic infomations about the page.  It can also be used as inter-page navigation.

## Examples

### Default \ Title \ Description \ TitleMode

1. Default
This is a standard page header with a back action.

2. With title and description
Users can add title and description for the page.

3. TitleMode
Both title and description is exist, user can specify the alignment of them.

### extra \ backIcon \ children \ style

1. extra
Users can custom own actions by extra. As follows, added an "Edit" action with Button element.

2. backIcon
Users can custom yourself icon instead of default.

3. children
As you know, It'll render below the title when there are children in PageHeader.

4. style and className
Users can add additional css class and style.
