## Tabs

Further classification of the content in the same view and make it easy to organize the content.

## Examples

There are some examples as follows which can help you get started quickly.

### default

By default, the props of type is "line" and size is "middle", you can use it simply as follows.

Note that the prop label is required in the component of Tabs.TabPane.

<Canvas>
  <Story name="basic">
    <Tabs>
      <Tabs.TabPane label="Tab-1">content one</Tabs.TabPane>
      <Tabs.TabPane label="Tab-2">content two</Tabs.TabPane>
      <Tabs.TabPane label="Tab-3">content three</Tabs.TabPane>
    </Tabs>
  </Story>
</Canvas>

### size

You can set the size of tabs.

<Canvas>
  <Story name="size">
    <Divider dashed textStyle={{ color: '#999' }}>
      small
    </Divider>
    <Tabs size="small">
      <Tabs.TabPane label="Tab-1">content one</Tabs.TabPane>
      <Tabs.TabPane label="Tab-2">content two</Tabs.TabPane>
      <Tabs.TabPane label="Tab-3">content three</Tabs.TabPane>
    </Tabs>
    <Divider dashed textStyle={{ color: '#999' }}>
      large
    </Divider>
    <Tabs size="large">
      <Tabs.TabPane label="Tab-1">content one</Tabs.TabPane>
      <Tabs.TabPane label="Tab-2">content two</Tabs.TabPane>
      <Tabs.TabPane label="Tab-3">content three</Tabs.TabPane>
    </Tabs>
  </Story>
</Canvas>
