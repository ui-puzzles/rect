## Avatar

Avatar can be used to display profile phote of users simply. It supports images, Icons, or letters.

## Examples

There are some examples as below which can help you get started quickly.

### default / shape / extra

1. default
By default, it'll get circle and the content can be text, Icon and image. when the content is text, the font size will automatically adjust to fit the content.

2. shape
Two options of "circle" and "square" are available for shape. By default the value is "circle".

3. extra
User can customize extra element what they want.
