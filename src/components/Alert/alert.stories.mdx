## Alert

Used to display some suggestive informations and attachs attention.

## Examples

There are some examples as below which can help you get started quickly.

### default

By default, the type is "info" and the alignment is "left".

<Canvas>
  <Story name="default">
    <Alert title="default Alert" />
  </Story>
</Canvas>

### type and border

Four options for type are available: 'info', 'warning', 'success' and 'danger'. Also can set the bordered = 'true' to display border or not.
